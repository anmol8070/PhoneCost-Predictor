Welcome to the Mobile Price Classification project! This project aims to classify mobile phones into different price categories based on their features using the K-Nearest Neighbors (KNN) algorithm.

Project Overview

The Mobile Price Classification project utilizes the KNN algorithm to predict the price category of a given mobile phone. The model has been trained using the Kaggle dataset of mobile phone features provided by Abhishek Sharma.

How it Works

To classify a mobile phone, you need to provide its features, such as battery power, RAM, storage, camera, etc., in the input form. The system will then use the trained KNN model to predict the appropriate price category for the given features.

Disclaimer

Please note that the predictions made by the system are based on the trained model and the provided dataset. The accuracy of the predictions may vary, and it is advisable to consider them as estimates rather than definitive values.

Get Started

To begin using the Mobile Price Classification system, head over to the "Classification" page and enter the relevant mobile features. The system will display the predicted price category based on the provided information.
